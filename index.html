Resumo CSS - Regras Básicas

1. Regra da Cascata

Ordem: importância → especificidade → último código vence.

2. Especificidade

  • Tag: menor peso
  • Classe: médio
  • ID: maior peso

3. Combinação de seletores

  • primeiro h1 → qualquer h1 dentro de .primeiro
  • primeira > h1 → apenas filhos diretos
  • primeira h1 + h1 → h1 logo após outro h1

4. Pseudo-classes

  • first-child → primeiro filho
  • last-child → último filho
  • nth-child(n) → filho posição n
  • nth-of-type(n) → n-ésimo do mesmo tipo

5. Display inline vs block

Inline: ocupa só o conteúdo, não aceita width/height.

Block: ocupa a linha inteira, aceita width/height e todas margens.

6. Padrão Inline e Block

Inline: a, span, strong, em

Block: div, p, h1-h6, section

7. Margin, Padding e Border

Margin: espaço externo | Padding: espaço interno | Border: contorno

Shorthand:

  • 1 valor = todos lados
  • 2 valores = vertical | horizontal
  • 3 valores = top | horizontal | bottom
  • 4 valores = top | right | bottom | left

8. Box-sizing

box-sizing: border-box; → largura inclui padding + border.
